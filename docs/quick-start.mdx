## Prerequisites

- **Python 3.7+** installed on your system
- **Docker** installed and running
- **Git** for cloning the repository
- **OpenAI API key** (optional, for AI analysis)

## Step 1: Clone and Setup

```bash
# Clone the repository
git clone https://github.com/colespicer/load-opt-agent.git
cd load-opt-agent

# Create virtual environment
python -m venv .venv

# Activate virtual environment
# On macOS/Linux:
source .venv/bin/activate
# On Windows:
.venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Step 2: Configure Environment

```bash
# Copy environment template
cp env.example .env

# Edit .env file and add your OpenAI API key (optional)
# OPENAI_API_KEY=your-api-key-here
```

<Callout type="info">
  **OpenAI API Key**: Optional for AI-powered analysis. Get one from [OpenAI Platform](https://platform.openai.com/api-keys).
</Callout>

## Step 3: Run Your First Test

```bash
# Run a comprehensive test (protocol + browser)
python run_test.py configs/pop_website_test.yaml
```

This will:
- Build Docker images (first time only)
- Run protocol-level load test with k6
- Run browser-level test with xk6-browser
- Generate comprehensive performance metrics
- Create interactive HTML reports
- Organize results in `output/`

## Step 4: View Results

```bash
# List all test results
python scripts/list_test_results.py list

# View latest results
python scripts/list_test_results.py latest

# Open the HTML report
open output/pop-website-2024-dev_azurewebsites_net/latest/load_test_report.html
```

## üö® Troubleshooting

### Common Issues

<Accordion>
  <AccordionItem title="Docker not running">
    ```bash
    # Start Docker Desktop
    open -a Docker  # macOS
    # Or start Docker service on Linux
    sudo systemctl start docker
    ```
  </AccordionItem>
  
  <AccordionItem title="Permission errors">
    ```bash
    # Ensure Docker permissions
    sudo usermod -aG docker $USER
    # Log out and back in
    ```
  </AccordionItem>
  
  <AccordionItem title="Memory issues with high-load tests">
    ```bash
    # Check available memory
    free -h
    # Reduce VUs or duration in config
    vus: 50  # Instead of 200
    duration: 5m  # Instead of 30m
    ```
  </AccordionItem>
  
  <AccordionItem title="Browser test timeouts">
    ```bash
    # Increase browser test timeouts
    browser_settings:
      timeout: "60s"  # Instead of 30s
    ```
  </AccordionItem>
  
  <AccordionItem title="OpenAI API errors">
    ```bash
    # Check your API key
    cat .env | grep OPENAI_API_KEY
    # Ensure key is valid and has credits
    ```
  </AccordionItem>
</Accordion>
