# Quick Start Guide

Get your first load test running with AI-powered analysis in under 5 minutes.

## Prerequisites

- **Python 3.7+** installed on your system
- **Docker** installed and running
- **Git** for cloning the repository
- **OpenAI API key** (optional, for AI analysis)

## üöÄ Step 1: Clone and Setup

```bash
# Clone the repository
git clone https://github.com/colespicer/load-opt-agent.git
cd load-opt-agent

# Create virtual environment
python -m venv .venv

# Activate virtual environment
# On macOS/Linux:
source .venv/bin/activate
# On Windows:
.venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## üîß Step 2: Configure Environment

```bash
# Copy environment template
cp env.example .env

# Edit .env file and add your OpenAI API key
# OPENAI_API_KEY=your-api-key-here
```

<Callout type="info">
  **OpenAI API Key**: Required for AI-powered analysis. Get one from [OpenAI Platform](https://platform.openai.com/api-keys).
</Callout>

## üéØ Step 3: Run Your First Test

```bash
# Test a sample API endpoint
python run_test.py configs/api_test.yaml
```

This will:
- ‚úÖ Build the Docker image (first time only)
- ‚úÖ Run a 1-minute load test with 100 virtual users
- ‚úÖ Generate comprehensive performance metrics
- ‚úÖ Create organized results in `output/`

## üö® Troubleshooting

### Common Issues

<Accordion>
  <AccordionItem title="Docker not running">
    ```bash
    # Start Docker Desktop
    open -a Docker  # macOS
    # Or start Docker service on Linux
    sudo systemctl start docker
    ```
  </AccordionItem>
  
  <AccordionItem title="Permission errors">
    ```bash
    # Ensure Docker permissions
    sudo usermod -aG docker $USER
    # Log out and back in
    ```
  </AccordionItem>
  
  <AccordionItem title="OpenAI API errors">
    ```bash
    # Check your API key
    cat .env | grep OPENAI_API_KEY
    # Ensure key is valid and has credits
    ```
  </AccordionItem>
</Accordion>
