# Test Results

The Load Testing & Optimization Agent generates comprehensive test results with multiple output formats and detailed analysis.

### File Types

<Accordion>
  <AccordionItem title="Raw Data Files">
    - **`summary_0.json`**: Raw k6 protocol metrics in JSONL format
    - **`playwright_results_0.json`**: Raw Playwright browser metrics
    - **Size**: 50MB-1GB depending on test duration and VUs
  </AccordionItem>
  
  <AccordionItem title="Analysis Files">
    - **`enhanced_analysis_report.json`**: Processed protocol metrics
    - **`browser_summary_analysis.json`**: Processed browser metrics
    - **Size**: 1-10MB depending on analysis depth
  </AccordionItem>
  
  <AccordionItem title="HTML Reports">
    - **`load_test_report.html`**: Comprehensive interactive report
    - **`manual_load_test_report.html`**: Fallback report

  </AccordionItem>
</Accordion>

## üîß Manual Report Generation

### Fallback Report

When the primary HTML generation fails, a manual report is created:

```python
# Manual report generation
def generate_manual_report(output_dir):
    enhanced_analysis = load_analysis_file('enhanced_analysis_report.json')
    browser_analysis = load_analysis_file('browser_summary_analysis.json')
    
    # Generate simple HTML from analysis files
    html = generate_html_from_analysis(enhanced_analysis, browser_analysis)
    
    # Save as manual_load_test_report.html
    write_html_file(output_dir, 'manual_load_test_report.html', html)
```

## üìä Data Formats

### JSONL Format (Protocol Data)

Protocol summary data uses JSONL format for efficient processing:

```json
{"type":"Metric","data":{"name":"vus","type":"gauge","contains":"default","thresholds":[],"submetrics":null},"metric":"vus"}
{"metric":"vus","type":"Point","data":{"time":"2025-08-12T05:37:27.201614783Z","value":0,"tags":{}}}
{"metric":"http_reqs","type":"Point","data":{"time":"2025-08-12T05:37:27.201614783Z","value":1,"tags":{}}}
```

### JSON Format (Playwright Browser Data)

Playwright browser data uses structured JSON format with comprehensive metrics:

```json
{
```
