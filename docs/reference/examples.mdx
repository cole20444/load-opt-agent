## ðŸ”— Running Examples

### Execute Examples
```bash
# Run basic example
python main.py configs/basic-test.yaml

# Run technology-specific example
python main.py configs/pop-website-test.yaml

# Run load testing scenario
python main.py configs/heavy-load.yaml
```

### Customize Examples
```bash
# Copy and modify
cp configs/basic-test.yaml configs/my-app.yaml

# Edit configuration
nano configs/my-app.yaml

# Run custom test
python main.py configs/my-app.yaml
```
