## üöÄ Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/colespicer/load-opt-agent.git
cd load-opt-agent

# 2. Set up environment
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
pip install -r requirements.txt

# 3. Configure environment (optional, for AI analysis)
cp env.example .env
# Add your OpenAI API key to .env

# 4. Run your first comprehensive test
python run_test.py configs/pop_website_test.yaml

# 5. View results
open output/pop-website-2024-dev_azurewebsites_net/latest/load_test_report.html
```
