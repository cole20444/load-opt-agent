# Interpreting Results

Learn how to understand and analyze your load test results for actionable insights.

## ðŸ“Š Understanding Your Results

### Result Files Overview
Each test run generates multiple files with different types of analysis:

- **`summary.json`** - Raw k6 metrics (21 metrics, 38K+ data points)
- **`test_report.json`** - Formatted summary with key metrics
- **`page_resource_analysis.json`** - Page resource performance analysis
- **`enhanced_analysis_report.json`** - Detailed performance breakdown
- **`ai_analysis_report.json`** - AI-generated optimization recommendations

## ðŸ” Key Metrics Explained

### Response Time Metrics
- **Average Response Time** - Mean response time across all requests
- **Median Response Time** - 50th percentile response time
- **P95 Response Time** - 95% of requests completed within this time
- **P99 Response Time** - 99% of requests completed within this time

### Throughput Metrics
- **Requests per Second (RPS)** - Number of requests processed per second
- **Total Requests** - Total number of requests made during the test
- **Successful Requests** - Requests that returned 2xx status codes
- **Failed Requests** - Requests that returned 4xx/5xx status codes

### Error Metrics
- **Error Rate** - Percentage of failed requests
- **HTTP Error Codes** - Breakdown of specific error types
- **Timeout Errors** - Requests that exceeded timeout limits

## ðŸŽ¯ Actionable Insights

### Performance Bottlenecks
1. **Server Processing** - Usually the biggest bottleneck
2. **Large Resources** - Unoptimized images and assets
3. **Network Issues** - DNS, CDN, or connection problems
4. **Database Queries** - Slow database operations

### Optimization Priorities
1. **High Priority** - Critical issues affecting user experience
2. **Medium Priority** - Significant optimization opportunities
3. **Low Priority** - Minor improvements and best practices

## ðŸ“Š Trend Analysis

### Comparing Test Runs
- **Historical data** - Compare performance over time
- **Before/After** - Measure optimization impact
- **Load patterns** - Understand performance under different loads

### Key Trends to Watch
- **Response time degradation** under increased load
- **Error rate increases** at higher concurrency
- **Resource loading patterns** across different pages
- **Performance improvements** after optimizations
